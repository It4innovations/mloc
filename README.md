# MLoC - Machine Learning on Cluster

Machine learning as a Service solution.

## Quickstart

```bash
python mloc/mloc.py
```

## Examples

### Dense Network

```bash
cd examples/dense
python dense.py
```

### Convolutional Network

```bash
cd examples/conv
python conv.py
```

## Documentation

To build documentation run `make html` in the [docs](./docs) directory.
