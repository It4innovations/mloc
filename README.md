# MLoC - Machine Learning on Cluster

Machine learning as a Service solution.

## Quickstart

```bash
python -m mloc
```

## Examples

### Dense Network

```bash
cd examples/dense
python dense.py
```

### Convolutional Network

```bash
cd examples/conv
python conv.py
```
